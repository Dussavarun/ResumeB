🚀 ResumeB — Modern Resume Builder

A full-stack resume builder that lets users create professional resumes with real-time preview, customizable sections, and downloadable templates.
Built using Next.js, Redux Toolkit, MongoDB, and Tailwind CSS.

Live Demo : https://resume-b-ohkj.vercel.app/

✨ Features
📄 Create professional resumes quickly

⚡ Real-time resume preview

🎨 Multiple resume templates

✏️ Editable sections (skills, projects, experience, certifications, etc.)

📱 Fully responsive design

💾 Save resume data to database

📥 Download resume as PDF

🔐 Authentication support

⚙️ Installation
**Clone the repository**: git clone https://github.com/Dussavarun/ResumeB.git

**Move into the project**: cd ResumeB

**Install dependencies** : npm install

**Run the development server** : npm run dev

**Open in browser** : http://localhost:3000


**🧠 How It Works**

1.User fills resume sections in the editor

2.Data is stored in Redux state

3.Resume preview updates instantly

4.Data is saved to MongoDB

5.Resume can be exported as a PDF
